>在python的世界观中，一切都是对象，一切变量都是指针
>python是弱类型语言，并非无类型
>python是解释性语言
>学py我更多是拿C/C++比较，站在C/C++角度去理解实现

```python
a = 1
a = "zhang"
a = [1,2,3]
```
- py中没有分号
- py中的缩进是有意义的，因为这里的代码块没有大括号，缩进和冒号就承担起了这个责任
- 注释用#，无多行注释
```python
for i in [1,2,3]:
	print(i)#注意前面的TAB
```
- 行内空格是无意义的
- 这里没有&&和|，这里有`and or`
- py中的分支控制是if elif else
- py中声明函数无需返回值类型，因为这是弱类型语言
- 声明函数用`def `
- 可以以这种奇特方式赋值
```python
def fibonacci(N):
	L = []
	a,b = 0,1
	while len(L)<N:
		a,b = b,a+b
		L.append(a)
	return L
```
- py中可以出现多返回值，分别用逗号隔开，用变量依次接收
```python
def real_imag_conj(val):
    return val.real, val.imag, val.conjugate()

r, i, c = real_imag_conj(3 + 4j)
print(r, i, c)
#输出：3.0 4.0 （3-4j)
```
- python的函数依旧是值传递，函数内部对副本操作
- python函数可以有默认参数值，可以在外部通过指定形参值，而在点明形参名时顺序不重要
```python
def fibonacci(N, a=0, b=1):#这里给出默认值，后续可以不传入
    L = []
    while len(L) < N:
        a, b = b, a + b
        L.append(a)
    return L
    
fibonacci(10)

fibonacci(10, a=1, b=2)
fibonacci(10,1,2)#这和上面这个相等

fibonacci(10,b=1,a=2)#这和上面的结果不同，真实传入形参的值
```
