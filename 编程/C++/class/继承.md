# 什么是继承
继承所讨论的对象是类（class），即一个类所拥有的属性会被其子类所继承，但子类所有的属性相应的更多，占用内存也更多。
同时，子类（如下面的Player）有两种类型，一个是本身自己的类型，一个是其父类型
子类在类型名后面加冒号和父类的类型名
```cpp
class Entity{
public:
	int X,Y;
	void move(int xa,int ya){
		X += xa;
		Y += ya;
	}
	private privatef(){};
}
class Player : public Entity
{
public:
	const char* name;
	void showname(){
		std::cout<<name<<std::endl;
	}
}
```
这里，Player就是Entity的public子类型，其拥有X，Y整数，也可调用move函数
但注意：**子类型无论如何都不能访问父类的private**

## 继承的三种类型
### 1、public继承
这种继承方式可以让子类内部访问父类所有public和protected函数和变量，但在外部只能通子类过访问父类的public
即：

| 父类      | ->  | 子类      |
| ------- | --- | ------- |
| public  | ->  | public  |
| protect | ->  | protect |
| private | ->  | 不可访问    |

```cpp
class parent{
public://public的变量与函数
	int X, Y;
	void pub_print(){
		std::cout<<"parent_pub"<<std::endl;
	}
protected://protected的变量与函数
	int a, b;
	void pro_print(){
		std::cout<<"parent_pro"<<std::endl;
	}
private://private的变量与函数
	int c, d;
	void pri_print(){
		std::cout<<"parent_pri"<<std::endl;
	}
};

class Child : public parent
{
public:
	Child(){}
	void c_print(){
		std::cout<<X+Y<<std::endl;//合法，内部访问
		std::cout<<b+a<<std::endl;//合法，内部访问
	//	std::cout<<c+d<<std::endl;//不合法，无访问权限
	}
	void inherent_print(){
		pub_print();//合法，内部调用
		pro_print();//合法，内部调用
	//	pri_print();//非法，无调用权限
	}
};
int main(){
	Child c;
	c.pub_print();//合法，外部调用
	//c.pro_print();//不合法，外部调用
	//c.pri_print();//不合法，无调用权限
	//变量同理哦
}
```



### 2、protected继承
>[!什么是protected标签]
`protected` 是 C++ 中**介于 `public`（公开）和 `private`（私有）之间的访问权限**，核心规则用一句话说：
**仅 “类自身、子类、同包（同一命名空间）” 可访问，类外部（如 `main` 函数）不可访问**。

这种继承会让子类只能在内部使用父类的public和protect，外部不能通过该子类访问

| 父类        | ->  | 子类        |
| --------- | --- | --------- |
| public    | ->  | protected |
| protected | ->  | protected |
| private   | ->  | 不可访问      |


```cpp
class parent{
public://public的变量与函数
	int X, Y;
	void pub_print(){
		std::cout<<"parent_pub"<<std::endl;
	}
protected://protected的变量与函数
	int a, b;
	void pro_print(){
		std::cout<<"parent_pro"<<std::endl;
	}
private://private的变量与函数
	int c, d;
	void pri_print(){
		std::cout<<"parent_pri"<<std::endl;
	}
};

class Child : public parent
{
public:
	Child(){}
	void c_print(){
		std::cout<<X+Y<<std::endl;//合法，内部访问
		std::cout<<b+a<<std::endl;//合法，内部访问
	//	std::cout<<c+d<<std::endl;//不合法，无访问权限
	}
	void inherent_print(){
		pub_print();//合法，内部调用
		pro_print();//合法，内部调用
	//	pri_print();//非法，无调用权限
	}
};
int main(){
	Child c;
	//c.pub_print();//不合法，外部调用
	//c.pro_print();//不合法，外部调用
	//c.pri_print();//不合法，无调用权限
	//变量同理哦
}
```

### 3、private继承
这种继承方式会使父类所有public和protected全部以private继承给子类

| 父类        | ->  | 子类      |
| --------- | --- | ------- |
| public    | ->  | private |
| protected | ->  | private |
| private   | ->  | 不可访问    |
