# 什么是this指针？
在class类中，我们有成员变量和函数，而当你用sizeof会发现这些函数是不占类的地址的。因为所有的类的实体共享一块代码段中的函数。这些函数想要知道具体要对那个实体操作，就需要this指针

在编译期，编译器会把调用函数的实体的地址用this指针传给被调用的函数
```cpp
class Entity{
public:
	int X, Y;
	Entity()
		:X(1),Y(2)
	{}
	void PlusX(int a){
		X += a;
	}
}
```
拿这个类举例，这段代码实际相当于：
```cpp
void PlusX(Entity* this, int a){
	this->X += a;
}//伪代码
```

- 若没有创建对象（`Test t;` 这一步），就没有 “对象地址” 可以传递给 `this`，此时 `this` 指针无任何意义，也不会存在。
- 补充：静态成员函数（`static` 修饰）不能使用 `this` 指针，因为它不需要对象就能调用（无对象地址可传递），也不能访问非静态成员变量。

**`this` 指针的 “存在周期”：仅在成员函数执行时有效**
