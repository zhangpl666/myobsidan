### 一、字体基础概念

LaTeX 字体由**系列（罗马 / 无衬线 / 等宽）、形状（直立 / 斜体 / 伪斜体）、重量（加粗 / 正常）三要素控制，需结合**宏包**和**命令**实现精细化调整。

### 二、核心字体命令（无宏包依赖，基础层）

#### 1. 字体系列（整体风格）

|命令|字体风格|示例效果|
|---|---|---|
|`\rmfamily`|罗马体（衬线体，默认）|`\rmfamily 这是罗马体示例`|
|`\sffamily`|无衬线体|`\sffamily 这是无衬线体示例`|
|`\ttfamily`|等宽体（打字机风格）|`\ttfamily 这是等宽体示例`|

#### 2. 字体样式（局部修饰）

|命令|样式效果|示例代码|
|---|---|---|
|`\textbf{文字}`|加粗|`\textbf{加粗文字}`|
|`\textit{文字}`|斜体|`\textit{斜体文字}`|
|`\underline{文字}`|下划线|`\underline{下划线文字}`|
|`\textsc{文字}`|小型大写字母|`\textsc{Small Caps}`（英文生效，中文无意义）|
|`\emph{文字}`|强调（默认斜体，可自定义）|`\emph{强调文字}`|

#### 3. 字体大小（预设层级）

LaTeX 预设了**9 级字号**，由小到大依次为：

latex

```latex
\tiny < \scriptsize < \footnotesize < \small < \normalsize（默认）< \large < \Large < \LARGE < \Huge
```

示例：

```latex
{\tiny 极小字号} {\Large 较大字号} {\Huge 超大字号}
```

#### 4. 自定义字号（精确控制）

用 `\fontsize{尺寸}{行距}\selectfont` 实现，单位为**磅（pt）**：

latex

```latex
\fontsize{12pt}{14pt}\selectfont 自定义字号（12磅字，14磅行距）
```

### 三、关键宏包（扩展字体能力）

#### 1. 通用字体宏包：`fontspec`（需 XeLaTeX/LuaLaTeX 编译）

用于调用系统本地字体（如 Windows 的 “微软雅黑”、macOS 的 “苹方”），支持中英文混排。

- 基础用法：

```latex
    \usepackage{fontspec} % 加载宏包
    \setmainfont{Times New Roman} % 设置西文主字体
    \setsansfont{Arial} % 设置西文无衬线字体
    \setmonofont{Courier New} % 设置西文等宽字体
```

- 中文支持（需配合 `ctex` 或直接设置中文字体）：
```latex
 \setCJKmainfont{微软雅黑} % 设置中文主字体（如微软雅黑）
 \setCJKsansfont{思源黑体} % 设置中文无衬线字体（如思源黑体）
    ```


#### 2. 中文专用宏包：`ctex` 系列

专为中文排版设计，自动处理中文字体、断行、标点等，是中文 LaTeX 的 “刚需”。

- 文档类选择：
```latex
   % 中文文章类（推荐）
   \documentclass{ctexart} 
   % 中文书籍类
   \documentclass{ctexbook}
   % 中文报告类
   \documentclass{ctexrep}
   ```

```latex
 \usepackage{ctex}
 \setCJKmainfont{宋体} % 中文正文字体
 \setCJKsansfont{黑体} % 中文无衬线字体
 \setCJKmonofont{Consolas} % 中文等宽字体（代码块常用）
 ```

#### 3. 数学字体宏包（扩展数学公式字体）

|宏包|效果说明|加载方式|
|---|---|---|
|`amsmath`|标准数学公式支持（必备）|`\usepackage{amsmath}`|
|`amssymb`|扩展数学符号（如积分、集合符号）|`\usepackage{amssymb}`|
|`mathpazo`|Palatino 风格数学字体|`\usepackage{mathpazo}`|
|`euscript`|花体数学符号（如\(\mathscr{A}\)）|`\usepackage{euscript}`|

#### 4. 特殊字体宏包

|宏包|功能说明|示例效果|
|---|---|---|
|`palatino`|Palatino 系列字体（西文优雅风格）|需配合 `\usepackage{palatino}`|
|`fourier`|Fourier 字体（数学与正文风格统一）|`\usepackage{fourier}`|
|`babel`|多语言支持（如英文、法文、德文）|`\usepackage[english]{babel}`|

### 四、中文字体特殊控制（`ctex` 宏包）

在 `ctex` 环境下，可通过**命令**或**宏包选项**调整中文字体：

#### 1. 全局字体设置（文档类选项）

```latex
% 中文文章类 + 指定字体为“黑体”
\documentclass[fontset=hei]{ctexart} 
% 可选 fontset：song（宋体，默认）、hei（黑体）、kai（楷体）、fs（仿宋）
```

#### 2. 局部字体切换

```latex
{\kaishu 楷体文字} {\heiti 黑体文字} {\songti 宋体文字} {\fangsong 仿宋文字}
```

### 五、常见问题与解决方案

|问题|原因|解决方案|
|---|---|---|
|中文字体不显示，编译报错|未加载 `ctex` 宏包或编译引擎不支持中文|改用 `ctexart` 文档类，或用 XeLaTeX 编译|
|字体命令（如`\textbf`）对中文无效|传统 pdflatex 对中文加粗支持有限|改用 XeLaTeX 并加载 `fontspec`，或用 `ctex` 宏包|
|数学公式字体与正文不统一|未加载数学字体宏包|加载 `mathpazo` 等宏包，或用 `\usepackage{amsmath}`|

### 六、综合示例（中文 + 西文 + 数学字体）

```latex
\documentclass{ctexart} % 中文文章类
\usepackage{amsmath, amssymb} % 数学公式支持
\usepackage{fontspec} % 西文字体自定义（需XeLaTeX编译）

% 西文字体设置
\setmainfont{Times New Roman} % 西文正文：Times New Roman
\setsansfont{Arial} % 西文无衬线：Arial
\setmonofont{Courier New} % 西文等宽：Courier New

% 中文字体（可选，ctex默认已适配）
\setCJKmainfont{微软雅黑} % 中文正文：微软雅黑

\begin{document}
% 字体系列示例
{\rmfamily 罗马体（西文）} {\sffamily 无衬线体（西文）} {\ttfamily 等宽体（西文）}

% 中文局部字体
{\kaishu 楷体中文} {\heiti 黑体中文}

% 字体样式
\textbf{加粗文字（中西文）} \textit{斜体文字（中西文）}

% 数学公式
\[
\textbf{y} = \mathbf{A} \cdot \textit{x} + \mathcal{B}, \quad x \in \mathbb{R}
\]
% \mathbf{加粗数学符号}，\mathcal{花体符号}，\mathbb{黑板粗体}

\end{document}
```
